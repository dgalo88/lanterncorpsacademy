Reglas del Juego
Cada clase posee un planeta base que se encuentra en un sector especifico del universo, el planeta base constituye para cada clase el punto de inicio del juego y servirá como refugio, el planeta base solo podrá ser ocupado por miembros del cuerpo de linternas correspondiente. Por ejemplo, el cuerpo de linternas verdes tienen como base el planeta Oa, en el sector 0. El planeta base del cuerpo de linternas rojas, Ysmault, se encuentra en el sector 666.
  1. Verde. Planeta Base Oa, Sector 0.
  2. Amarillo. Planeta Base Qward, Sector -1 (“Universo Antimateria”).
  3. Negro. Planeta Base El Mundo Muerto de Ryut, Sector 665.
  4. Rojo. Planeta Base Ysmault, Sector 666.
  5. Azul. Planeta Base Odym, Sector 1.
  6. Indigo. Planeta Base: desconocido, Sector 3.
  7. Violeta. Planeta Base Zamaron, Sector 1412.
Todos los colores tienen como característica en común que sus poderes se manifiestan mediante anillos de poder de tecnología alienígena avanzada, alimentados por una batería en el planeta base de cada organización. Este anillo constituye el único artículo o item realmente equipado/portado por un jugador.
